# Fontra Pak for Windows
## Version 2026.3.3.16

A browser-based font editor for your desktop.

---

## 🚀 Quick Start

1. **Double-click** `Fontra Pak.exe`
2. Fontra Pak will launch automatically in your default browser
3. No Python installation required!

---

## 📋 System Requirements

- **Windows 10** or later
- Modern web browser (Chrome, Firefox, Edge)
- **No Python required!**

---

## 📁 Supported File Formats

### Read & Write
- `.ufo` - Unified Font Object
- `.designspace` - Designspace documents
- `.fontra` - Fontra format
- `.rcjk` - RoboCJK

### Partial Support
- `.glyphs` - Glyphs format
- `.glyphspackage` - Glyphs package

### Read Only
- `.ttf` - TrueType fonts
- `.otf` - OpenType fonts
- `.woff` / `.woff2` - Web fonts
- `.ttx` - XML font format

---

## 🎨 Features

- **Visual glyph editing** - Edit letter shapes with precision tools
- **Axis manipulation** - Work with variable font axes
- **Component management** - Build glyphs from reusable components
- **Anchors & Guidelines** - Precise positioning tools
- **Text preview** - See your fonts in action
- **Local processing** - Your fonts stay on your computer
- **Tools menu positioning** - Toggle tools between Top/Bottom

---

## 🔧 Troubleshooting

### Windows SmartScreen Warning
Click "More info" → "Run anyway" (this is normal for unsigned executables)

### Browser doesn't open
Manually open: http://localhost:8000

### Port already in use
Close other instances or use a different port in settings
